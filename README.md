# 🐍 Python 期末考试 4.5 小时速成计划

> **专为 C 语言背景学习者定制的 Python 速成教材**

## 📚 项目简介

本教材专门为有 C 语言基础的学习者设计，通过**类比学习**的方式，帮助你在 4.5 小时内快速掌握 Python 核心知识，顺利通过期末考试。

### 🎯 核心策略

- ✅ **类比学习**：始终将 Python 特性与 C 语言对比，快速理解异同
- ✅ **放弃深究**：暂时不纠结底层实现，专注语法和应用
- ✅ **实践为主**：每个知识点配备代码示例，立即实践
- ✅ **抓大放小**：重点掌握核心数据结构（列表、字典）和文件操作

## 🎓 学习建议

### ✅ 推荐做法

1. **按顺序学习**：严格按照第一部分 → 第二部分 → 第三部分 → 第四部分的顺序
2. **边学边练**：每看完一个知识点，立即运行对应的示例代码
3. **对比思考**：遇到新概念时，主动思考"这在 C 语言中是怎么实现的？"
4. **查阅速查表**：学习过程中随时翻阅 `quick_reference.md`
5. **完成练习题**：每部分学完后，完成对应的练习题

### ❌ 避免做法

1. **不要跳跃学习**：基础不牢，地动山摇
2. **不要只看不练**：编程是实践技能，必须动手写代码
3. **不要纠结细节**：暂时不要深究 Python 的内存管理、GIL 等底层机制
4. **不要忽视错误**：查看 `common_mistakes.md`，避免常见陷阱
